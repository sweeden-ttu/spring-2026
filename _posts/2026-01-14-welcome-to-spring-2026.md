Welcome to the Spring 2026 coursework site for Software Verification and Validation (CS-5374) and Cryptography (CS-6343) at Texas Tech University. This site contains course materials, lecture notes, and resources organized by topic and week.

<!--more-->

## About This Site

This site provides comprehensive course materials and resources for graduate-level courses in software engineering. All content is organized using the jekyll-TeXt-theme for easy navigation and publication-quality presentation.

## Courses Offered

### Software Verification and Validation (CS-5374)

**Instructors**: Akbar Siami Namin, Ph.D. (Primary), Hasan Al-Qudah (Co-Instructor)

This course covers:
- Software testing fundamentals and strategies
- Black box and white box testing
- Model-based and graph-based testing
- Formal verification and model checking
- LangSmith for LLM testing and evaluation
- AI/LLM/RL evaluation techniques

**Course Page**: [View Course Introduction]({{ '/software-verification/introduction/' | relative_url }})

### Cryptography (CS-6343)

This course covers cryptographic methods and security protocols.
