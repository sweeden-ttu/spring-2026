# Introduction to Software Verification and Validation (CS-5374)

## Course Overview

**Software Verification and Validation (CS-5374)** is a graduate-level course covering methods and techniques for verifying and validating software systems. This course emphasizes both theoretical foundations and practical applications, with hands-on experience using modern tools for testing and evaluation.

### Course Information

- **Course Code**: CS-5374
- **Course Name**: Software Verification and Validation
- **Semester**: Spring 2026
- **Start Date**: January 14, 2026
- **End Date**: May 15, 2026
- **Time Zone**: America/Chicago

### Instructors

- **Primary Instructor**: Akbar Siami Namin, Ph.D.
- **Co-Instructor**: Hasan Al-Qudah

## Course Objectives

By the end of this course, students will be able to:

1. **Understand V&V Fundamentals**
   - Define verification and validation
   - Distinguish between different V&V techniques
   - Apply V&V throughout software lifecycle

2. **Master Testing Techniques**
   - Black box and white box testing
   - Model-based testing
   - Graph-based and fault-based testing
   - Security testing

3. **Apply Formal Methods**
   - Formal verification techniques
   - Model checking
   - Specification analysis

4. **Use Modern Tools**
   - LangSmith for LLM testing and evaluation
   - Open source tools for AI/LLM/RL testing
   - Testing frameworks and debugging tools

5. **Evaluate and Validate**
   - Test adequacy criteria
   - Coverage analysis
   - Performance evaluation
   - Quality assurance
