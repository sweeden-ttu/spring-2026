# Topic 5: White Box Testing

## Overview

White box testing (also known as structural testing or code-based testing) is a testing method where the internal structure of the software is known and used to design test cases.

## Learning Objectives

- Understand white box testing principles
- Apply code coverage criteria
- Design test cases from code structure
- Use white box testing tools

## Key Concepts

- **Statement Coverage**: Testing every executable statement
- **Branch Coverage**: Testing every decision branch
- **Path Coverage**: Testing execution paths
- **Condition Coverage**: Testing boolean conditions
- **Loop Testing**: Testing loop structures
