## Course Overview

**CS-6343 Cryptography** covers cryptographic methods and security protocols for secure communication, data protection, and digital signatures.

### Topics Covered

<div style="display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 1rem; margin: 1rem 0;">

<div style="border-left: 3px solid #333; padding-left: 1rem;">
  <h4 style="margin: 0 0 0.5rem 0; color: #000;">Symmetric Cryptography</h4>
  <ul style="margin: 0; padding-left: 1.2rem; font-size: 0.9rem; color: #666;">
    <li>Block ciphers (AES, DES)</li>
    <li>Stream ciphers</li>
    <li>Modes of operation</li>
  </ul>
</div>

<div style="border-left: 3px solid #333; padding-left: 1rem;">
  <h4 style="margin: 0 0 0.5rem 0; color: #000;">Asymmetric Cryptography</h4>
  <ul style="margin: 0; padding-left: 1.2rem; font-size: 0.9rem; color: #666;">
    <li>RSA algorithm</li>
    <li>Elliptic curve cryptography</li>
    <li>Key exchange (Diffie-Hellman)</li>
  </ul>
</div>

<div style="border-left: 3px solid #333; padding-left: 1rem;">
  <h4 style="margin: 0 0 0.5rem 0; color: #000;">Applications</h4>
  <ul style="margin: 0; padding-left: 1.2rem; font-size: 0.9rem; color: #666;">
    <li>Digital signatures</li>
    <li>Hash functions</li>
    <li>Authentication protocols</li>
  </ul>
</div>

</div>
